🟢 Checkpoint X: Complete Content Hierarchy Bulk Import System (Subjects, Chapters & Topics)

Goal:

Extend the Content Editor bulk creation functionality so that Administrators and Super Administrators can bulk import not only Subjects, but also Chapters and Topics using downloadable CSV templates and the same bulk upload/import workflow currently used in:

**Admin Dashboard → User Management → Bulk User Import**

The implementation must provide a consistent, scalable, production-ready content management experience that supports rapid creation of large content hierarchies without manual entry.

---

# Context

Currently:

### Supported

Content Editor supports:

* Subject Creation
* Bulk Subject Import

---

### Missing

There is no equivalent bulk import workflow for:

* Chapters
* Topics

As a result:

* Admins must create Chapters manually.
* Admins must create Topics manually.
* Large content structures require excessive repetitive work.
* Content onboarding becomes slow and inefficient.

---

# New Requirement

The Content Editor should support bulk import for:

### Level 1

Subjects

### Level 2

Chapters

### Level 3

Topics

using:

* Download CSV Template
* Upload CSV
* Validation
* Import Preview
* Bulk Creation
* Error Reporting

following the same workflow and user experience as:

**User Management → Bulk User Import**

---

# Important Requirement

The implementation should create a complete hierarchy import framework.

Administrators should be able to:

### Option A

Import Subjects separately.

### Option B

Import Chapters separately.

### Option C

Import Topics separately.

### Option D (Recommended)

Import an entire hierarchy using properly linked CSV files.

Example:

Subject → Physics

Chapter → Mechanics

Topic → Newton's Laws
