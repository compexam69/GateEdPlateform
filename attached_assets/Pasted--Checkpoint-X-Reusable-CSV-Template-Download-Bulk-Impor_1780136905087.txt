🟢 Checkpoint X: Reusable CSV Template Download & Bulk Import Framework for Question Import and Subject Creation

Goal:

Implement the same downloadable CSV template file functionality and bulk import workflow currently used in the Bulk User Import feature within the Admin Dashboard and extend it to:

1. Question Import (Quiz & Question Editor)
2. Subject Creation (Content Editor / Lecture Management)

The implementation must reuse the existing Bulk User Import architecture, user experience, validation patterns, template generation approach, error handling, and import workflow wherever possible to ensure consistency across the platform.

---

Context

Currently:

Existing Feature

User Management → Bulk User Import

Already supports:

* Download CSV Template
* Upload CSV
* Validation
* Error reporting
* Import processing
* Success feedback

This workflow is familiar to administrators and already works well.

---

Missing Features

The same functionality does not currently exist for:

Quiz & Question Editor

Administrators cannot:

* Download a question import template
* Generate import-ready CSV files easily
* Follow a consistent import workflow

---

Content Editor

Administrators cannot:

* Download a subject creation template
* Bulk create subjects
* Use the same import experience available in User Management

---

New Requirement

Implement identical workflow patterns used in Bulk User Import for:

Quiz & Question Editor

Add:

* Download Question CSV Template
* Upload Question CSV
* Validate Question CSV
* Import Questions

---

Content Editor

Add:

* Download Subject CSV Template
* Upload Subject CSV
* Validate Subject CSV
* Bulk Create Subjects

---

Important Requirement

Reuse existing Bulk User Import architecture wherever possible.

Avoid:

* Building separate import systems
* Duplicating logic
* Creating inconsistent user experiences

The workflow should feel identical throughout the Admin Dashboard.
