🟢 Checkpoint 17: Move Super Admin Self-Edit Actions Inside Main Profile Edit Popup

Goal: Redesign the Super Admin profile editing workflow by moving the Edit/Pencil buttons for Name, Email, and Mobile Number inside the main Profile Edit/Pencil popup/modal under a dedicated “Edit Details” section while keeping the actual details visible on the Profile page.

Context: Currently:

* The profile photo has a centralized Pencil/Edit button with a popup/modal workflow.
* The Name, Email, and Mobile Number fields are displayed on the Profile page with separate visible Edit/Pencil buttons beside them.

This creates UI clutter and inconsistent editing behavior across the Profile module.

Important Permission Rule:

* Only the Super Admin has permission to edit their own:

  * Name
  * Email
  * Mobile Number
* Admins cannot edit their own details.
* Students/regular users cannot edit their own immutable details.

New Requirement:

* Apply this redesign ONLY for the Super Admin Profile Settings page.
* Keep the Name, Email, and Mobile Number visible normally on the Profile page.
* Remove ONLY the inline Edit/Pencil buttons beside these fields from the Profile page UI.
* Move these edit actions inside the main Profile Edit/Pencil popup/modal.
* Add a dedicated “Edit Details” section inside the popup/modal containing:

  * Edit Name option
  * Edit Email option
  * Edit Mobile Number option

Instructions:

1. Analyze Existing Profile Editing Workflow:

* Review:

  * Super Admin Profile Settings UI
  * Existing inline Edit/Pencil buttons
  * Profile Edit/Pencil popup/modal
  * Existing permission logic
  * Existing edit dialogs/forms
  * Responsive/mobile behavior
* Identify how the current edit actions are rendered and triggered.

2. Redesign Super Admin Profile Edit Structure:

* Keep the following details visible normally on the Profile page:

  * Name
  * Email
  * Mobile Number
* Remove ONLY the visible inline Edit/Pencil buttons beside these fields.
* Inside the main Profile Edit/Pencil popup/modal:

  * Add a clearly separated:

    * “Edit Details”

  * Edit Name action
