🟢 Checkpoint X: Focus Timer Module Full-Screen Non-Scrollable Layout Optimization (Mobile, Desktop & PWA)

Goal:

Redesign and optimize the Focus Timer module so that all essential content, controls, statistics, and timer functionality are visible within a single screen viewport without requiring vertical scrolling.

This applies to:

* Focus Timer (Hamburger Menu Navigation)
* Focus Module (Bottom Navigation on Mobile)

Important:

These are the same module and must maintain identical behavior, layout logic, state management, and functionality.

The page should be effectively non-scrollable during normal usage and provide a complete "at-a-glance" focus experience.

---

### Context

Based on the attached screenshots:

#### Mobile Issues

Current mobile layout shows:

* Timer mode buttons
* Topic selector
* Large timer circle

However:

* Play button
* Reset button
* Session statistics
* Daily streak cards
* Focus summary section

are pushed below the viewport and require scrolling.

This creates several usability issues:

* Users cannot immediately access timer controls.
* Users cannot view session metrics while focusing.
* Critical controls are hidden.
* Focus workflow becomes fragmented.
* Excessive scrolling reduces productivity.

---

#### Desktop Issues

Desktop view also shows:

* Large unused empty vertical spacing.
* Excessive whitespace beneath the timer.
* Statistics positioned too far below.
* Content hierarchy is inefficient.

Although desktop scrolling is less problematic, the current layout wastes significant screen real estate.

---

### New Requirement

The Focus Timer module must be redesigned so that:

#### Mobile

Everything important is visible immediately:

* Timer mode selector
* Topic selector
* Timer
* Play button
* Reset button
* Session statistics
* Daily streak summary

without scrolling.

---

#### Desktop

The layout should:

* Use available viewport height efficiently.
* Reduce unnecessary whitespace.
* Keep all key information above the fold.
* Present a more professional dashboard layout.
